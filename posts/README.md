## 37. 배포란 무엇인가?

- 배포란 사용자에게 프로그램을 제공하는 방법
- 서버용 컴퓨터를 구성하는 것은 여러 어려움이 있음
- 따라서 가장 간단한 방법인 클라우드 컴퓨터를 주로 이용, 강의에서는 AWS를 사용
- 서버용 컴퓨터로 리눅스를 사용하는 이유
    - 무료 오픈소스
    - 우수한 보안성
    - 구조 안정성
